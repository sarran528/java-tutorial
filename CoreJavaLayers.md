# Core Java Layered System

This document outlines the essential layers and topics required for a comprehensive understanding of Core Java. Mastery of each layer is critical for building robust Java applications, especially in backend development.

---

## 1. Language Foundations
- JVM, JRE, JDK architecture
- Compilation vs interpretation
- Bytecode execution model
- Data types (primitive vs reference)
- Variables, scope, lifetime
- Type casting (implicit, explicit)
- Operators (arithmetic, logical, bitwise)

## 2. Control Flow
- if / else / switch
- loops (for, while, do-while)
- break, continue, return
- nested flow structures

## 3. Methods
- Method definition and invocation
- Parameters vs arguments
- Pass-by-value (critical concept)
- Method overloading
- Recursion

## 4. Object-Oriented Programming (Core Layer)
- Class and object
- Constructors (default, parameterized)
- this keyword
- Object lifecycle

**Pillars:**
- Encapsulation
- Inheritance
- Polymorphism (compile-time vs runtime)
- Abstraction (abstract class vs interface)

## 5. Advanced OOP Concepts
- Method overriding
- Dynamic method dispatch
- super keyword
- final keyword (class, method, variable)
- Object class methods (toString, equals, hashCode)

## 6. Packages and Access Control
- Package creation and usage
- import statements
- Access modifiers:
  - public
  - protected
  - default
  - private

## 7. Strings and Immutable Objects
- String vs StringBuilder vs StringBuffer
- String pool concept
- Immutability
- Common string operations

## 8. Arrays
- Single-dimensional arrays
- Multi-dimensional arrays
- Memory behavior
- Arrays utility class

## 9. Exception Handling
- Checked vs unchecked exceptions
- try, catch, finally
- throw vs throws
- Custom exceptions
- Exception hierarchy

## 10. Collections Framework (Critical)
- List (ArrayList, LinkedList)
- Set (HashSet, TreeSet)
- Map (HashMap, TreeMap)
- Queue, Deque
- Iterators
- Comparable vs Comparator
- Internal working (hashing basics)

## 11. Generics
- Type safety
- Generic classes and methods
- Wildcards (?, extends, super)

## 12. Java I/O (File Handling)
- File class
- Byte streams vs character streams
- Buffered streams
- Reading/writing files
- Serialization and deserialization

## 13. Multithreading & Concurrency
- Thread lifecycle
- Creating threads (Thread vs Runnable)
- Synchronization
- Race conditions
- Inter-thread communication
- Executor framework (basic)

## 14. Lambda Expressions & Functional Interfaces
- Functional interfaces
- Lambda syntax
- Method references
- Stream API (map, filter, reduce)

## 15. Memory Management
- Stack vs heap
- Garbage collection
- Reference types
- Memory leaks (logical, not manual)

## 16. Inner Classes
- Member inner class
- Static nested class
- Anonymous class
- Local inner class

## 17. Enums and Annotations
- Enum basics and use cases
- Built-in annotations (@Override, etc.)
- Custom annotations (basic understanding)

## 18. Reflection (Awareness Level)
- Class metadata
- Inspecting methods/fields
- Dynamic invocation

## 19. Java Time API
- LocalDate, LocalTime, LocalDateTime
- Formatting and parsing

## 20. Basic Networking (Optional Core Edge)
- Sockets (basic idea)
- URL and HTTP basics

## 21. JVM Internals (Conceptual)
- ClassLoader
- Execution engine
- JIT compiler
- Memory areas (heap, stack, metaspace)

---

### Structural Reality
- 1–4 → mandatory base
- 5–10 → real programming competence
- 11–14 → modern Java capability
- 15–21 → system-level understanding

> Skipping order creates shallow knowledge that fails in backend work like Spring.
